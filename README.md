Public Issue & Incident Dashboard adalah dashboard interaktif yang dibangun menggunakan Power BI untuk menganalisis data pengaduan masyarakat dan insiden sosial. Dashboard ini bertujuan membantu pemerintah daerah, instansi terkait, dan stakeholder publik dalam memantau isu sosial, mengevaluasi dampak terhadap masyarakat, serta menentukan prioritas penanganan berbasis data.

Dashboard mengintegrasikan informasi terkait jenis isu sosial, lokasi kejadian, tingkat keparahan (severity), status penyelesaian, durasi penyelesaian, serta dampak demografi masyarakat.
